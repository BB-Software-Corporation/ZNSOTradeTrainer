# ZNSOTradeTrainer
这是我们小宝宝软件公司的第二个出品，这个软件可以模拟交易，不需要像富途牛牛那样手机号码开户来模拟。
## ⭐功能亮点
* 这个软件里面使用了模拟账户登录的方法来区分开操作
* 允许用户以K线图的方法来看到自己的交易记录
